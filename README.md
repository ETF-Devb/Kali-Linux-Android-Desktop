# Kali-Linux-Android-Desktop
Run a full Kali Linux Desktop GUI on Android! This professional script automates Kali NetHunter installation with a hardware-accelerated XFCE4 graphical interface via Termux-X11. Includes Android 12+ phantom process fixes, custom Bubblewrap (bwrap) patches for sandboxed apps, and core Kali tools. Quick, stable, and optimized.
